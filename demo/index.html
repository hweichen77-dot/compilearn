Pointers

Every variable in your program lives somewhere in memory. A pointer is simply a value that holds where something is — its address — rather than the thing itself. Once you can pass an address around, you can share and mutate data without copying it.

Reading a pointer is called dereferencing: follow the address to the value it points at. In Python you rarely do this by hand, but the model underneath every list, object, and function call is exactly this — names bound to locations.
